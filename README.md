NovaStore
NovaStore is a modern and minimalistic e-commerce web application built with React, JavaScript, and CSS. The project is currently a work in progress and aims to deliver a smooth and stylish shopping experience.

At this stage, the application integrates a product API to display a dynamic product list. The goal is to expand its functionality by adding categories, a shopping cart, and a checkout process. The layout is fully responsive and designed with clean UI principles, focusing on usability and aesthetics.

Further improvements will include user authentication, product filters, and enhanced UX features.
